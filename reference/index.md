# Package index

## genieBPC Functions

Functions for abstracting and manipulating GENIE BPC data

- [`synapse_version()`](https://genie-bpc.github.io/genieBPC/reference/synapse_version.md)
  : Return list of available GENIE BPC data releases
- [`pull_data_synapse()`](https://genie-bpc.github.io/genieBPC/reference/pull_data_synapse.md)
  : Obtain clinical & genomic data files for GENIE BPC Project
- [`create_analytic_cohort()`](https://genie-bpc.github.io/genieBPC/reference/create_analytic_cohort.md)
  : Select cohort of patients for analysis
- [`select_unique_ngs()`](https://genie-bpc.github.io/genieBPC/reference/select_unique_ngs.md)
  : Selecting corresponding unique next generation sequencing reports
- [`drug_regimen_sunburst()`](https://genie-bpc.github.io/genieBPC/reference/drug_regimen_sunburst.md)
  : Visualize drug regimen sequences in a sunburst plot
- [`check_genie_access()`](https://genie-bpc.github.io/genieBPC/reference/check_genie_access.md)
  : Check Access to GENIE Data
- [`drug_regimen_list`](https://genie-bpc.github.io/genieBPC/reference/drug_regimen_list.md)
  : List of Drug Regimen Names by Cohort
- [`genie_panels`](https://genie-bpc.github.io/genieBPC/reference/genie_panels.md)
  : Genomic Panels Included in GENIE BPC Data
- [`cohort_institution`](https://genie-bpc.github.io/genieBPC/reference/cohort_institution.md)
  : List of Institutions by Cohort
- [`nsclc_test_data`](https://genie-bpc.github.io/genieBPC/reference/nsclc_test_data.md)
  : Simulated fake GENIE BPC data for function examples and tests
- [`regimen_abbreviations`](https://genie-bpc.github.io/genieBPC/reference/regimen_abbreviations.md)
  : List of Drug Regimen Abbreviations
- [`set_synapse_credentials()`](https://genie-bpc.github.io/genieBPC/reference/set_synapse_credentials.md)
  : Connect to 'Synapse' API
- [`synapse_tables`](https://genie-bpc.github.io/genieBPC/reference/synapse_tables.md)
  : 'Synapse' table IDs

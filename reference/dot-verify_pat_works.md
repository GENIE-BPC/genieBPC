# Verify a synapse Personal Access Token works

Verify a synapse Personal Access Token works

## Usage

``` r
.verify_pat_works(pat)
```

## Arguments

- pat:

  'Synapse' Personal Access Token. If NULL, package will search package
  environment for "pat".

## Value

a 'Synapse' token

## Examples

``` r
if (FALSE) { # \dontrun{
.verify_pat_works(pat = "test")
} # }
```
